# 🎬 RacerFlix

RacerFlix is my first front-end project, inspired by modern streaming platforms like Netflix, focused on delivering an immersive experience for racing and automotive content.

## 🚀 Features

- 🎥 Background video with dynamic themes (lightdark mode)
- 👤 Interactive profile system with video avatars
- 🎬 Hover video previews on cards
- ⏳ Custom loading screen with animation
- 🧭 Global header component (reusable across pages)
- 💡 Theme toggle with localStorage persistence
- 📱 Responsive layout

## 🛠️ Technologies

- HTML5 (semantic structure)
- CSS3 (animations, layout, effects)
- JavaScript (DOM manipulation, state management)

## 🎯 About the Project

This project was built as my first front-end experience, focusing on learning how to create interactive interfaces and replicate real-world platforms.

It demonstrates skills in UIUX design, performance optimization, and dynamic component creation using pure JavaScript.
